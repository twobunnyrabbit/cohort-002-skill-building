## Quick Start

Clone the skill building repository:

```bash
git clone git@github.com:ai-hero-dev/cohort-002-skill-building.git
```

Clone the project repository:

```bash
git clone git@github.com:ai-hero-dev/cohort-002-project.git
```

---

## Two Types of Activities

In this cohort, we're going to be doing two types of activities: skill building or project work.

Each day starts with skill building, where we do some problem and solution exercises and some explainers to build the skills we need. Then in the project work section, we go and apply those skills.

This should give a really nice balance of working on the skills individually and then applying them in a creative way.
